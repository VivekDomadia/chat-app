# Node.js Chat App

A web Chat room where people can send message, location to group of people.

[(◔‿◔)!☞ Live Demo](https://nodejs-chat-app.vivekdomadia11.repl.co/)

#### Platforms / Technologies

* Node.js

* Bootstrap

* Socke.io

## ⌨️ Development

Setup:

- run `npm install`
- run `npm start`
